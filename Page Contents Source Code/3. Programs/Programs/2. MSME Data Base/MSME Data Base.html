MSME Data Base


In Sri Lanka, there is no proper single database developed by any institution to capture all MSME data. Lack of such dedicated MSME database is a serious issue in developing of strategic/policy decisions in the industry sector. Having identified this lacuna, NEDA established an MSME Data Base aiming to enter the information of all entrepreneurs in Sri Lanka. This will allow the development officers of NEDA to have the data of Entrepreneurs in soft format and update them periodically. Additionally, we believe this endeavour will stimulate research and development in the industry sector as well.


The main highlighted Objectives of establishing MSME database

  • Information dissemination through printed and electronic material
  • Digitalizing NEDA activities
  • Ensuring Effective response to the needs of the Enterprises
  • Providing market linkages facilities for the SMEs

Scope

This system will consist of mainly two subsystems: a web application and a mobile application. NEDA officers, regional officers of each district secretariat division, resource persons (authorized individuals for catering intellectual resources to the entrepreneurs), entrepreneurs, and the general public will be the stakeholders of the system. The mobile application will enable regional officers to gather information about entrepreneurs via a forum and submit as requests for the reference of NEDA officers. Once these requests are confirmed and approved by the NEDA, the database will be updated. The web application will have a portal to view and download data from the backend database. Also, it will have a login option for NEDA officers to access the system and add/update/delete records.

Functions

  1. Mobile Application
    The mobile application is for the use of regional officers in each district secretariat division. The application will support an online forum feature to collect details about entrepreneurs and the forum should be sent as a request for the reference of NEDA officers and If any information of an entrepreneur needs to be updated, regional officers can send them as requests. If an existing entrepreneur is shutting down his/her business, regional officers can inform those via the application.
  2. Web Application
    Web Application Authorized NEDA officers can visit the website, filter, and view details about entrepreneurs, download data in SPSS compatible format. Each authorized NEDA officer will have a unique username and password to login to the system and, add/update/delete records according to the requests from regional officers.
